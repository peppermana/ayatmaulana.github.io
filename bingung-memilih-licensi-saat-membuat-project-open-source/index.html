Bingung memilih Licensi saat membuat project Open Source?

2017-09-17

Linux

tak jarang diantara kita merasa kebingungan ketika membuat project open source akan menggunakan lisenci apa entah itu GPL, MIT, APACHE, dll

itu wajar sobat karena kita takut kalo project tersebut besar dan suda banyak orang yang menggunakan project tersebut kita salah

sebaiknya sobat bisa simak beberapa lisence open source yang populer dan digunakan di github di bawah ini

  1. example :Hexo, Laravel, Lumen, CodeIgniter, Bootstrap, Vuejs dll

  2. Apache Lisence

    example : Weex, Casandra, Hadoop, Ant

  3. GNU GPLv3 ( General Public Lisence )

  4. BSD Lisence

    example: Facebook ReactJs , Facebook React Native, PHP, Redis

  5. Ecslipe Plugin

  6. Mozilla Public Lisence
