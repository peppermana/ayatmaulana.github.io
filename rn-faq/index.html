Apa itu React Native ?

React Native merupakan suatu tools yang dikembangkan oleh Facebook, yang mana kita dapat membuat aplikasi Android maupun iOS dalam sekali koding, ditulis menggunakan Bahasa Javascript ES6 & JSX, selengkapnya React Native Â· A framework for building native apps using React

Kenapa harus react native?

  • Mudah dipelajar
  • Disupport oleh facebook
  • Dukungan Library yang banyak
  • Benar benar hasil akhirnya native, tidak seperti hybrid, let say Cordova, Ionic

Apa benefit yang saya dapatkan ketika belajar react native ?

berikut yang akan kamu dapatkan ketika belajar react native

  1. Efesiensi Waktu & Koding
  2. Sekali Bekerja dua hasil
  3. Tidak perlu belajar native programming language, let say Java, Objective C, Swift

Apa saja tools yang dibutuhkan

Kamu harus membawa laptop sendiri dan Android device sendiri, dan pastikan dilaptop sudah terinstall

  • Android Studio
  • Java JDK 8
  • NodeJS v8.5
  • Code Editor (VS Code)
  • Vysor (Opsional)
